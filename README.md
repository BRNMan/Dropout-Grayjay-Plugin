# Grayjay Plugin Template

## Development

1.  install typescript `tsc` and `node/npm`, `deno` or, `bun`
2.  `npm update`
3.  `tsc`
4.  `npm run dev:node`
5.  once you load the plugin into the app the home screen should have a fake video that links to
    <https://example.com>
6.  tap it to open the video
